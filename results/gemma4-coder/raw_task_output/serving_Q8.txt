===== REASONING (thought channel) =====
Goal: implement `longest_palindromic_substring(s)` in Python and state time/space complexity.

Edge cases: empty string → "", length 1 → the string itself, all same chars → the whole string, palindrome at ends/middle.

Approach: expand around center. For each index (odd center) and each gap between indices (even center), expand outward while characters match. Track the best start/end seen.

Complexity: O(n²) time (n centers, O(n) expansion per center), O(1) extra space (excluding output).


===== ANSWER (content) =====
Expand from each possible center (single character for odd lengths, pair of characters for even lengths) and grow outward as long as the substring remains a palindrome. The longest valid expansion is kept.

```python
def longest_palindromic_substring(s):
    if not s:
        return ""

    start, end = 0, 0

    def expand_around_center(left, right):
        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        return left + 1, right - 1

    for i in range(len(s)):
        l1, r1 = expand_around_center(i, i)
        if r1 - l1 > end - start:
            start, end = l1, r1

        l2, r2 = expand_around_center(i, i + 1)
        if r2 - l2 > end - start:
            start, end = l2, r2

    return s[start : end + 1]
```

Time complexity: O(n²), Space complexity: O(1)
