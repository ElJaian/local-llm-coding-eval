===== REASONING (thought channel) =====
Goal: implement `longest_palindromic_substring(s)` returning the longest palindromic contiguous substring of `s`, plus state time/space complexity in one line.

Edge cases: empty string (return ""), single char (return s), all same chars (return s), even vs odd length palindromes.

Approach: expand-around-center. For each index (and each gap between indices), expand outward while chars match. Track the best start/length.

Complexity: O(n²) time (n centers, avg O(n) expansion), O(1) extra space beyond output.


===== ANSWER (content) =====
Expand from each possible palindrome center (every index and every gap between indices) and track the best match.

```python
def longest_palindromic_substring(s):
    if not s:
        return ""

    start, length = 0, 1

    def expand(left, right):
        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        return left + 1, right - left - 1

    for i in range(len(s)):
        # Odd length (center at s[i])
        s1, l1 = expand(i, i)
        if l1 > length:
            start, length = s1, l1

        # Even length (center between s[i] and s[i+1])
        s2, l2 = expand(i, i + 1)
        if l2 > length:
            start, length = s2, l2

    return s[start:start + length]
```

Time: O(n²) | Space: O(1) (excluding output)
